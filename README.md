# Paydunk Front End Plugin

### Usage:

```js
<script type="text/javascript" src="jquery.paydunk.js"></script> 
<script>
$('#paydunkButton').paydunk({
    appID        : '7k2f9w57LmFfzU7eP3AQEZW19H5qj8XRwEE73VfB', //required
    price        : 20, //required
    order_number : 1,
    tax          : 0,
    shipping     : 0
});
</script>
```
